== Set environment variables

**Run this in the Terminal tab to set up your session — all module examples use these variables:**

[source,bash,subs="attributes+"]
----
export LITELLM_API_BASE="{litellm_api_base_url}"
export LITELLM_API_KEY="{litellm_virtual_key}"
----

== Verify your connection

[source,bash]
----
curl $LITELLM_API_BASE/models \
  -H "Authorization: Bearer $LITELLM_API_KEY" \
  -s | python3 -c "import sys,json; [print(m['id']) for m in json.load(sys.stdin)['data']]"
----

Expected output — the models your key has access to:

[subs="attributes+"]
----
{litellm_available_models_list}
----

NOTE: If your key includes *Gemini 2.5 Pro*, always set `max_tokens` to at least *1000* — it is a thinking model that uses reasoning tokens before generating output.
