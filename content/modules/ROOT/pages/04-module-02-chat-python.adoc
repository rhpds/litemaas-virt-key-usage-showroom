= Chat with Python
:navtitle: Chat with Python

The OpenAI Python SDK works directly with LiteMaaS — just point it at your API base URL.
No new SDK to install.

== Learning objectives

By the end of this module, you'll be able to:

* Use the OpenAI Python SDK with LiteMaaS
* Stream responses in Python
* Build a simple multi-turn conversation loop
* Integrate with Langchain

TIP: Examples use `granite-3-2-8b-instruct` as the default. Swap the `model` field with any chat model in your key: `{litellm_available_models_list}`

== Exercise 1: Basic chat with the OpenAI SDK

[source,python,subs="attributes+"]
----
import os
from openai import OpenAI

client = OpenAI(
    base_url="{litellm_api_base_url}",
    api_key="{litellm_virtual_key}"
)

response = client.chat.completions.create(
    model="granite-3-2-8b-instruct",
    messages=[
        {"role": "user", "content": "What is Red Hat OpenShift in one sentence?"}
    ],
    max_tokens=100
)

print(response.choices[0].message.content)
----

Run it:

[source,bash]
----
python3 chat_basic.py
----

=== Verify

✓ You see a one-sentence description of OpenShift printed to the terminal.

== Exercise 2: Streaming

[source,python,subs="attributes+"]
----
from openai import OpenAI

client = OpenAI(
    base_url="{litellm_api_base_url}",
    api_key="{litellm_virtual_key}"
)

stream = client.chat.completions.create(
    model="granite-3-2-8b-instruct",
    messages=[{"role": "user", "content": "Explain containers in 3 sentences"}],
    stream=True
)

for chunk in stream:
    if chunk.choices[0].delta.content:
        print(chunk.choices[0].delta.content, end="", flush=True)
print()
----

=== Verify

✓ Text appears progressively, token by token.

== Exercise 3: Multi-turn conversation loop

[source,python,subs="attributes+"]
----
from openai import OpenAI

client = OpenAI(
    base_url="{litellm_api_base_url}",
    api_key="{litellm_virtual_key}"
)

messages = [
    {"role": "system", "content": "You are a concise Red Hat technical assistant."}
]

print("Chat with the AI (type 'quit' to exit)\n")

while True:
    user_input = input("You: ").strip()
    if user_input.lower() == "quit":
        break

    messages.append({"role": "user", "content": user_input})

    response = client.chat.completions.create(
        model="granite-3-2-8b-instruct",
        messages=messages,
        max_tokens=300
    )

    reply = response.choices[0].message.content
    messages.append({"role": "assistant", "content": reply})
    print(f"AI: {reply}\n")
----

=== Verify

✓ The assistant maintains context across multiple turns.

== Exercise 4: Langchain integration

[source,python,subs="attributes+"]
----
from langchain_openai import ChatOpenAI
from langchain_core.messages import HumanMessage, SystemMessage

llm = ChatOpenAI(
    model="granite-3-2-8b-instruct",
    openai_api_base="{litellm_api_base_url}",
    openai_api_key="{litellm_virtual_key}",
    max_tokens=200
)

messages = [
    SystemMessage(content="You are a helpful Red Hat technical assistant."),
    HumanMessage(content="What is the difference between a pod and a deployment in Kubernetes?")
]

response = llm.invoke(messages)
print(response.content)
----

=== Verify

✓ Langchain correctly routes the request through LiteMaaS and returns a response.
