== LiteMaaS API Credentials

[%autowidth,cols="a,a"]
|===
| *API Base URL* | `{litellm_api_base_url}`
| *Virtual Key* | `{litellm_virtual_key}`
| *Available Models* | `{litellm_available_models_list}`
|===

Run this in the terminal to set your environment variables — all module examples use them:

[source,bash,subs="attributes+"]
----
export LITELLM_API_BASE="{litellm_api_base_url}"
export LITELLM_API_KEY="{litellm_virtual_key}"
----
