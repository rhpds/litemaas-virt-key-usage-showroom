= RHDP MaaS — Using AI Models
:navtitle: RHDP MaaS — AI Model Access

Welcome! This guide shows you how to use the Red Hat Demo Platform Model-as-a-Service (RHDP MaaS) — your access point to production AI models running on OpenShift AI.

Your virtual key gives you an OpenAI-compatible API endpoint backed by real GPU-hosted models.
