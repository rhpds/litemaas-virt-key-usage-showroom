= Welcome to RHDP MaaS
:navtitle: Welcome

RHDP Model-as-a-Service (MaaS) gives you access to production AI models through a single OpenAI-compatible API. Your virtual key is scoped to the models you need — no infrastructure to manage.

== What you'll learn

* Call AI models with curl and Python
* Use embeddings for semantic search
* Count tokens before sending requests
* Convert documents to Markdown with Docling
