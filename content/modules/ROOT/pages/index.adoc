= RHDP MaaS — Using AI Models
:navtitle: RHDP MaaS — AI Model Access

Welcome! This guide shows you how to use the Red Hat Demo Platform Model-as-a-Service (RHDP MaaS) — your access point to production AI models running on OpenShift AI.

Your virtual key gives you an OpenAI-compatible API endpoint backed by real GPU-hosted models. Whether you're building a RAG pipeline, experimenting with embeddings, or calling a chat model from a lab, this guide has working examples in curl, Python, and AgnosticV.

== What you'll learn

* How to authenticate and list available models
* How to call chat models with curl and Python
* How to generate text embeddings for semantic search and RAG
* How to count tokens before sending a request
* How to convert documents to Markdown using Docling (if enabled)
* How to wire LiteMaaS virtual keys into an AgnosticV catalog
